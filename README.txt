Instrukcja:

1) Edytuj 'script.js' ustawiając BOT_NAME, INVITE_LINK, SUPPORT_LINK, oraz API_STATUS_URL (opcjonalne).
2) Wgraj folder 'bot_website' na hosting statyczny (GitHub Pages, Netlify, Vercel) lub serwer WWW.
3) Jeśli chcesz panel z OAuth, daj znać — mogę przerobić stronę na React + proste OAuth.

Pliki zawarte w paczce:
- index.html
- styles.css
- script.js
- logo.svg
- README.txt

Powodzenia!
